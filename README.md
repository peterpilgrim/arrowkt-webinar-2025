## Setup

  * JDK 24+, Kotlin 2.2
  * Gradle, SDKman
  * IntelliJ IDEA (or your preferred IDE)
  * GitHub repo (starter code provided https://github.com/peterpilgrim/arrowkt-webinar-2025)


## Build

Clean the distribution

```zsh
gradle clean
```

Build the code

```zsh
gradle build
```


Execute the tests

```zsh
gradle test
```
